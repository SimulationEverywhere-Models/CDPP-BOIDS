Para llevar a cabo las pruebas ejecutar en Windows los distintos archivos .bat

boids1.bat	Vuelo de dos pajaros

boids2.bat	Dos pajaros forman un grupo

boids3.bat	Dos pajaros enfrentados y otro que los sigue

boids4.bat	Cuatro pajaros

boids5.bat	Dos pajaros, uno modifica su velocidad para formar un grupo
